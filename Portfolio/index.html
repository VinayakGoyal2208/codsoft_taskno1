Steganography: Hiding text into Image

This project uses C# within the .NET framework to implement steganography by embedding secret text in images via the Least Significant Bit (LSB) method. It allows users to encode and decode hidden messages in image files while preserving the visual integrity. The project leverages .NET's image processing libraries and optional cryptographic tools for enhanced security.
